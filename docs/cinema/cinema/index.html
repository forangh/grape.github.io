这是一个最好的时代。

很多时候我们都在想，中国电影的未来究竟是什么样的？

没有人能比我们更了解电影行业的高速发展所面临的机遇与挑战。我们当下要做的事就是服务好电影人与电影院，才能夯实已经取得的成绩，让电影事业顺利进入下一个成长阶段，非常期待与各位师兄共同建设电影行业一小时经济圈。
